# Web Platform (`apps/web-platform`)

**The Surface Story**: The user-facing interface for the KRYPTO-VELAMEN Instrument.

## Tech Stack
- **Framework**: Next.js 14+ (React)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion (for "Glitch" and "Reveal" effects)
- **State**: Zustand / React Query

## Core Features (MVP)
1.  **The Archive Grid**: A visual browser for the `research/` and `drafts/` corpus.
2.  **Fragment Editor**: A markdown-aware editor with "Dial" controls.
3.  **Terminal Mode**: A CLI-themed UI for "Digital-First" consumption.
